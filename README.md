## Instalación

Instale previamente el  [Node.js](https://nodejs.org/) v10+.

Instale las dependencias y e inicie el servidor.

```sh
cd npm nodevars
```
```sh
cd npm install
```
```sh
cd ng serve
```

Salu2.
